# 01.用户认证

用户认证负责注册、登录、验证码、邀请码校验。
本篇讲登录链路，也就是用户输入账号密码后，后端如何校验用户、生成 JWT，并让前端保存登录状态。

## 1.完整链路

```text
POST /api/v1/user/login
  -> Router
  -> controller
  -> service
  -> dao
  -> MySQL
  -> service 生成 JWT
  -> controller 返回 token
  -> frontend 保存 token
```

对应到项目文件：

```text
GoNexus/frontend/src/pages/AuthPage.tsx
  -> GoNexus/router/user.go
  -> GoNexus/controller/user/user.go
  -> GoNexus/service/user/user.go
  -> GoNexus/dao/user/user.go
  -> GoNexus/common/mysql/mysql.go
  -> GoNexus/model/user.go
  -> GoNexus/utils/myjwt/jwt.go
  -> GoNexus/frontend/src/store/authStore.ts
```

## 2.POST：前端发起登录请求

文件：`GoNexus/frontend/src/pages/AuthPage.tsx`

```tsx
// 登录走 /user/login，注册走 /user/register
const endpoint = isLogin ? "/user/login" : "/user/register"

// 登录只需要账号和密码；注册需要邮箱、密码、验证码
const payload = isLogin ? { username, password } : { email, password, captcha }

// 发起请求到后端
const res = await axios.post(apiUrl(endpoint), payload)
```

登录时实际发送的是：

```tsx
axios.post(apiUrl("/user/login"), {
  // 用户输入的登录账号
  username,

  // 用户输入的登录密码
  password,
})
```

### 登录 controller

```go
func Login(c *gin.Context) {
	req := new(LoginRequest)
	res := new(LoginResponse)

	// 把前端 JSON 参数绑定到 req
	if err := c.ShouldBindJSON(req); err != nil {
		c.JSON(http.StatusOK, res.CodeOf(code.CodeInvalidParams))
		return
	}

	// 调用 service 校验账号密码，并生成 token
	token, code_ := user.Login(req.Username, req.Password)
	if code_ != code.CodeSuccess {
		c.JSON(http.StatusOK, res.CodeOf(code_))
		return
	}

	// 登录成功后，把 token 返回给前端
	res.Success()
	res.Token = token
	c.JSON(http.StatusOK, res)
}
```

## 5.Service：校验账号和密码

文件：`GoNexus/service/user/user.go`

```go
func Login(username, password string) (string, code.Code) {
	var userInformation *model.User
	var ok bool

	// 先查用户是否存在
	if ok, userInformation = user.IsExistUser(username); !ok {
		return "", code.CodeUserNotExist
	}

	// 对前端传来的密码做 MD5 后，与数据库密码比较
	if userInformation.Password != utils.MD5(password) {
		return "", code.CodeInvalidPassword
	}

	// 账号密码正确，生成 JWT token
	token, err := myjwt.GenerateToken(userInformation.ID, userInformation.Username)
	if err != nil {
		return "", code.CodeServerBusy
	}

	// 返回 token 给 controller
	return token, code.CodeSuccess
}
```

	// 根据 username 查询用户

	}

## 10.JWT 工具：生成 token

文件：`GoNexus/utils/myjwt/jwt.go`

```go
func GenerateToken(id uint, username string) (string, error) {
	claims := MyClaims{
		// 用户 ID
		ID: id,

		// 用户名
		Username: username,

		RegisteredClaims: jwt.RegisteredClaims{
			// token 签发者
			Issuer: config.GetConfig().Issuer,

			// token 主题
			Subject: config.GetConfig().Subject,

			// token 过期时间
			ExpiresAt: jwt.NewNumericDate(time.Now().Add(time.Hour * time.Duration(config.GetConfig().ExpireDuration))),

			// token 签发时间
			IssuedAt: jwt.NewNumericDate(time.Now()),
		},
	}

	// 使用 HS256 签名算法创建 token
	token := jwt.NewWithClaims(jwt.SigningMethodHS256, claims)

	// 使用配置里的 key 签名并返回字符串 token
	return token.SignedString([]byte(config.GetConfig().Key))
}
```
