# 04.RAG知识库链路

RAG 知识库负责把上传文档切分、向量化并写入 Redis Stack。
聊天选择 RAG 模型时，后端会先从 Redis 检索相关片段，再拼接 prompt 调用 LLM。

## 1.完整链路

```text
KnowledgeBase.tsx
  -> fileApi.upload
  -> POST /api/v1/file/upload
  -> Router
  -> JWT middleware
  -> controller
  -> service 保存 uploads/{username}
  -> RAGIndexer
  -> extractPlainText
  -> splitText
  -> embedding
  -> Redis Stack 向量索引

聊天时选择 RAG 模型：
AIHelper
  -> AliRAGModel
  -> RAGQuery
  -> Redis 检索相关 chunk
  -> BuildRAGPrompt
  -> LLM
```

对应到项目文件：

```text
GoNexus/frontend/src/components/chat/KnowledgeBase.tsx
  -> GoNexus/frontend/src/pages/KnowledgePage.tsx
  -> GoNexus/frontend/src/api/file.ts
  -> GoNexus/router/File.go
  -> GoNexus/controller/file/file.go
  -> GoNexus/service/file/file.go
  -> GoNexus/common/rag/rag.go
  -> GoNexus/common/rag/split.go
  -> GoNexus/common/redis/redis.go
  -> GoNexus/common/aihelper/model.go
```

```go
func NewRAGIndexer(indexID string) (*RAGIndexer, error) {
	ctx := context.Background()

	// 创建 embedding 模型
	embedder, err := NewRAGEmbedder(ctx)
	if err != nil {
		return nil, err
	}

	// 初始化 Redis Stack 向量索引
	dimension := config.GetConfig().GetEmbeddingDimension()
	if err := redisPkg.InitRedisIndex(ctx, indexID, dimension); err != nil {
		return nil, fmt.Errorf("failed to init redis index: %w", err)
	}
```

```go
indexerConfig := &redisIndexer.IndexerConfig{
	Client:    rdb,
	KeyPrefix: redis.GenerateIndexNamePrefix(indexID),
	BatchSize: 10,

	// 把每个文档 chunk 转成 Redis Hash
	DocumentToHashes: func(ctx context.Context, doc *schema.Document) (*redisIndexer.Hashes, error) {
		return &redisIndexer.Hashes{
			Key: fmt.Sprintf("%s:%s", indexID, doc.ID),
			Field2Value: map[string]redisIndexer.FieldValue{
				// content 字段会被 embedding 成 vector
				"content": {Value: doc.Content, EmbedKey: "vector"},

				// metadata 保存来源文件
				"metadata": {Value: source},
			},
		}, nil
	},
}

// 绑定 embedding 模型
indexerConfig.Embedding = embedder
```

## 10.RAG：读取文件并切 chunk

文件：`GoNexus/common/rag/rag.go`

```go
func (r *RAGIndexer) IndexFile(ctx context.Context, filePath string) error {
	// 读取 txt/md 文本内容
	content, err := extractPlainText(filePath)
	if err != nil {
		return err
	}

	// 切成多个 chunk
	chunks := splitText(content, defaultChunkSize, defaultChunkOverlap)
	if len(chunks) == 0 {
		return fmt.Errorf("file has no indexable content: %s", filePath)
	}
```

```go
docs := make([]*schema.Document, 0, len(chunks))
for i, chunk := range chunks {
	docs = append(docs, &schema.Document{
		// 每个 chunk 一个文档 ID
		ID: fmt.Sprintf("%s_chunk_%d", fileID, i),

		// chunk 文本内容
		Content: chunk,

		// 保存来源文件和 chunk 序号
		MetaData: map[string]any{
			"source": filePath,
			"chunk":  i,
		},
	})
}

// 写入 Redis 向量索引
_, err = r.indexer.Store(ctx, docs)
return err
```

## 12.RAG 聊天：检索文档并拼 prompt

文件：`GoNexus/common/aihelper/model.go`

```go
func (o *AliRAGModel) StreamResponse(ctx context.Context, messages []*schema.Message, cb StreamCallback) (string, error) {
	// 取用户最新问题
	if len(messages) == 0 {
		return "", fmt.Errorf("empty messages")
	}
	query := messages[len(messages)-1].Content

	// 创建当前用户的 RAG 查询器
	ragQuery, err := rag.NewRAGQuery(ctx, o.username)
	if err != nil {
		return o.streamWithoutRAG(ctx, messages, cb)
	}

	// 从 Redis 检索相关文档片段
	docs, err := ragQuery.RetrieveDocuments(ctx, query)
	if err != nil {
		return o.streamWithoutRAG(ctx, messages, cb)
	}

	// 把检索结果和用户问题拼成最终 prompt
	ragPrompt := rag.BuildRAGPrompt(query, docs)
```

```go
// 复制历史消息
ragMessages := make([]*schema.Message, len(messages))
copy(ragMessages, messages)

// 用 RAG prompt 替换最后一条用户问题
ragMessages[len(ragMessages)-1] = &schema.Message{
	Role:    schema.User,
	Content: ragPrompt,
}

// 使用拼好的 RAG prompt 调用 LLM 流式输出
stream, err := o.llm.Stream(ctx, ragMessages)
if err != nil {
	return "", fmt.Errorf("ali rag stream failed: %v", err)
}
```

## 13.RAG：构建最终 prompt

文件：`GoNexus/common/rag/rag.go`

```go
func BuildRAGPrompt(query string, docs []*schema.Document) string {
	if len(docs) == 0 {
		// 没检索到内容时，直接使用原问题
		return query
	}

	contextText := ""
	for i, doc := range docs {
		// 把检索到的 chunk 拼成参考文档
		contextText += fmt.Sprintf("[文档 %d]: %s\n\n", i+1, doc.Content)
	}

	// 把参考文档和用户问题合成最终 prompt
	prompt := fmt.Sprintf(`...`, contextText, query)
	return prompt
}
```
