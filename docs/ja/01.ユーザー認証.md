# 01. ユーザー認証

ユーザー認証は、登録、ログイン、Captcha、招待コード検証を担当します。
この章ではログインの流れ、つまりユーザーがアカウントとパスワードを入力したあと、バックエンドがどのように検証し、JWT を生成し、フロントエンドにログイン状態を保存させるかを説明します。

## 1. 全体の流れ

```text
POST /api/v1/user/login
  -> Router
  -> controller
  -> service
  -> dao
  -> MySQL
  -> service が JWT を生成
  -> controller が token を返す
  -> frontend が token を保存
```

対応するファイル：

```text
GoNexus/frontend/src/pages/AuthPage.tsx
  -> GoNexus/router/user.go
  -> GoNexus/controller/user/user.go
  -> GoNexus/service/user/user.go
  -> GoNexus/dao/user/user.go
  -> GoNexus/common/mysql/mysql.go
  -> GoNexus/model/user.go
  -> GoNexus/utils/myjwt/jwt.go
  -> GoNexus/frontend/src/store/authStore.ts
```

## 2. POST：フロントエンドがログインリクエストを送る

ファイル：`GoNexus/frontend/src/pages/AuthPage.tsx`

```tsx
// ログインは /user/login、登録は /user/register
const endpoint = isLogin ? "/user/login" : "/user/register"

// ログインは username と password、登録は email、password、captcha を使う
const payload = isLogin ? { username, password } : { email, password, captcha }

// バックエンドへリクエストを送る
const res = await axios.post(apiUrl(endpoint), payload)
```

ログイン時に実際に送られる内容：

```tsx
axios.post(apiUrl("/user/login"), {
  // ユーザーが入力したログインアカウント
  username,

  // ユーザーが入力したログインパスワード
  password,
})
```

## 4. Controller：パラメータをバインドして service を呼ぶ

ファイル：`GoNexus/controller/user/user.go`

```go
type LoginRequest struct {
	// ログインアカウント
	Username string `json:"username"`

	// ログインパスワード
	Password string `json:"password"`
}
```

```go
func Login(c *gin.Context) {
	req := new(LoginRequest)
	res := new(LoginResponse)

	// フロントエンドの JSON を req にバインドする
	if err := c.ShouldBindJSON(req); err != nil {
		c.JSON(http.StatusOK, res.CodeOf(code.CodeInvalidParams))
		return
	}

	// service でアカウントとパスワードを検証し、token を生成する
	token, code_ := user.Login(req.Username, req.Password)
	if code_ != code.CodeSuccess {
		c.JSON(http.StatusOK, res.CodeOf(code_))
		return
	}

	// ログイン成功後、token をフロントエンドへ返す
	res.Success()
	res.Token = token
	c.JSON(http.StatusOK, res)
}
```

## 5. Service：アカウントとパスワードを検証する

ファイル：`GoNexus/service/user/user.go`

```go
func Login(username, password string) (string, code.Code) {
	var userInformation *model.User
	var ok bool

	// ユーザーが存在するか確認する
	if ok, userInformation = user.IsExistUser(username); !ok {
		return "", code.CodeUserNotExist
	}

	// 入力パスワードを MD5 化し、DB のパスワードと比較する
	if userInformation.Password != utils.MD5(password) {
		return "", code.CodeInvalidPassword
	}

	// 認証成功後、JWT token を生成する
	token, err := myjwt.GenerateToken(userInformation.ID, userInformation.Username)
	if err != nil {
		return "", code.CodeServerBusy
	}

	// controller へ token を返す
	return token, code.CodeSuccess
}
```

```go
func IsExistUser(username string) (bool, *model.User) {
	// username でユーザーを検索する
	user, err := mysql.GetUserByUsername(username)
	if err != nil {
		return false, nil
	}

	// 検索成功時、ユーザー情報を返す
	return true, user
}
```

## 7. MySQL：実際に DB へアクセスする

```go
func GetUserByUsername(username string) (*model.User, error) {
	user := new(model.User)

	// GORM で username が一致する最初のレコードを検索する
	err := DB.Where("username = ?", username).First(user).Error
	return user, err
}
```

同等の SQL：

```sql

SELECT * FROM users WHERE username = ? LIMIT 1;
```

## 10. JWT ツール：token を生成する

ファイル：`GoNexus/utils/myjwt/jwt.go`

```go
func GenerateToken(id uint, username string) (string, error) {
	claims := MyClaims{
		// ユーザー ID
		ID: id,

		// ユーザー名
		Username: username,

		RegisteredClaims: jwt.RegisteredClaims{
			// token 発行者
			Issuer: config.GetConfig().Issuer,

			// token の subject
			Subject: config.GetConfig().Subject,

			// token の有効期限
			ExpiresAt: jwt.NewNumericDate(time.Now().Add(time.Hour * time.Duration(config.GetConfig().ExpireDuration))),

			// token 発行時刻
			IssuedAt: jwt.NewNumericDate(time.Now()),
		},
	}

	// HS256 署名アルゴリズムで token を作成する
	token := jwt.NewWithClaims(jwt.SigningMethodHS256, claims)

	// 設定の key で署名し、文字列 token を返す
	return token.SignedString([]byte(config.GetConfig().Key))
}
```
