# 04. RAG ナレッジベース連携

RAG ナレッジベースは、アップロードされた文書を chunk に分割し、embedding して Redis Stack に保存します。
チャットで RAG モデルを選択すると、バックエンドは Redis から関連 chunk を検索し、prompt を組み立てて LLM を呼び出します。

## 1. 全体の流れ

```text
KnowledgeBase.tsx
  -> fileApi.upload
  -> POST /api/v1/file/upload
  -> Router
  -> JWT middleware
  -> controller
  -> service が uploads/{username} に保存
  -> RAGIndexer
  -> extractPlainText
  -> splitText
  -> embedding
  -> Redis Stack ベクトルインデックス

RAG モデルでチャットする時：
AIHelper
  -> AliRAGModel
  -> RAGQuery
  -> Redis から関連 chunk を検索
  -> BuildRAGPrompt
  -> LLM
```

対応するファイル：

```text
GoNexus/frontend/src/pages/KnowledgePage.tsx
  -> GoNexus/frontend/src/components/chat/KnowledgeBase.tsx
  -> GoNexus/frontend/src/api/file.ts
  -> GoNexus/router/File.go
  -> GoNexus/controller/file/file.go
  -> GoNexus/service/file/file.go
  -> GoNexus/common/rag/rag.go
  -> GoNexus/common/rag/split.go
  -> GoNexus/common/redis/redis.go
  -> GoNexus/common/aihelper/model.go
```

```go
func UploadRagFile(c *gin.Context) {
	res := new(UploadFileResponse)

	// multipart form の file フィールドを読む
	uploadedFile, err := c.FormFile("file")
	if err != nil {
		c.JSON(http.StatusOK, res.CodeOf(code.CodeInvalidParams))
		return
	}

	// JWT middleware の context から現在ユーザーを取得する
	username := c.GetString("userName")
	if username == "" {
		c.JSON(http.StatusOK, res.CodeOf(code.CodeInvalidToken))
		return
	}

	// service でファイル検証、保存、RAG index 作成を行う
	filePath, err := file.UploadRagFile(username, uploadedFile)
	if err != nil {
		c.JSON(http.StatusOK, res.CodeOf(code.CodeServerBusy))
		return
	}

	// 保存パスを返す
	res.Success()
	res.FilePath = filePath
	c.JSON(http.StatusOK, res)
}
```

## 5. Service：uploads/{username} に保存する

ファイル：`GoNexus/service/file/file.go`

```go
func UploadRagFile(username string, file *multipart.FileHeader) (string, error) {
	// ファイル種別とファイル名を検証する
	if err := utils.ValidateFile(file); err != nil {
		return "", err
	}

	// ユーザーディレクトリ uploads/{username} を作成する
	userDir := filepath.Join("uploads", username)
	if err := os.MkdirAll(userDir, 0755); err != nil {
		return "", err
	}

	// ユーザーごとに 1 つの共通ナレッジインデックスを使う
	indexID := rag.UserKnowledgeIndexID(username)

	// UUID でサーバー側ファイル名を生成する
	uuid := utils.GenerateUUID()
	ext := filepath.Ext(file.Filename)
	filename := uuid + ext
	filePath := filepath.Join(userDir, filename)
```

```go
// アップロードファイルを開く
src, err := file.Open()
if err != nil {
	return "", err
}
defer src.Close()

// 保存先ファイルを作成する
dst, err := os.Create(filePath)
if err != nil {
	return "", err
}
defer dst.Close()

// uploads/{username}/{uuid}.ext へコピーする
if _, err := io.Copy(dst, src); err != nil {
	return "", err
}
```

## 7. RAG：ファイルを読み chunk に分割する

ファイル：`GoNexus/common/rag/rag.go`、`GoNexus/common/rag/split.go`

```go
func (r *RAGIndexer) IndexFile(ctx context.Context, filePath string) error {
	// txt/md のテキスト内容を読み取る
	content, err := extractPlainText(filePath)
	if err != nil {
		return err
	}

	// 複数 chunk に分割する
	chunks := splitText(content, defaultChunkSize, defaultChunkOverlap)
	if len(chunks) == 0 {
		return fmt.Errorf("file has no indexable content: %s", filePath)
	}
```

```go
// Redis ベクトルインデックスへ保存する
_, err = r.indexer.Store(ctx, docs)
return err
```

```go
const (
	// 各 chunk の最大長
	defaultChunkSize = 800

	// 隣接 chunk 間で残す重複コンテキスト
	defaultChunkOverlap = 100
)
```

## 8. RAG Chat：検索して prompt を作る

ファイル：`GoNexus/common/aihelper/model.go`

```go
func (o *AliRAGModel) StreamResponse(ctx context.Context, messages []*schema.Message, cb StreamCallback) (string, error) {
	// 最新のユーザー質問を取得する
	query := messages[len(messages)-1].Content

	// 現在ユーザーの RAG 検索器を作成する
	ragQuery, err := rag.NewRAGQuery(ctx, o.username)
	if err != nil {
		return o.streamWithoutRAG(ctx, messages, cb)
	}

	// Redis から関連文書 chunk を検索する
	docs, err := ragQuery.RetrieveDocuments(ctx, query)
	if err != nil {
		return o.streamWithoutRAG(ctx, messages, cb)
	}

	// 最終 RAG prompt を作成する
	ragPrompt := rag.BuildRAGPrompt(query, docs)
```

```go
// 最後のユーザーメッセージを RAG prompt に置き換える
```
