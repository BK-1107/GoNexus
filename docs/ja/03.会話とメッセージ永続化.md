# 03. 会話とメッセージ永続化

会話とメッセージ永続化は、ユーザーメッセージと AI の返信を保存します。
この章では、AIHelper がまずメモリ上のコンテキストへ保存し、その後 RabbitMQ 経由で非同期に MySQL へ書き込む流れを説明します。

## 1. 全体の流れ

```text
AIHelper.AddMessage
  -> メモリコンテキスト a.messages
  -> saveFunc
  -> GenerateMessageMQParam
  -> RabbitMQ Publish
  -> MQMessage consumer
  -> dao/message
  -> MySQL
```

対応するファイル：

```text
GoNexus/common/aihelper/aihelper.go
  -> GoNexus/common/rabbitmq/meesage.go
  -> GoNexus/common/rabbitmq/rabbitmq.go
  -> GoNexus/dao/message/message.go
  -> GoNexus/common/mysql/mysql.go
  -> GoNexus/model/message.go
```

## 2. AIHelper：保存コールバックを作成する

ファイル：`GoNexus/common/aihelper/aihelper.go`

```go
func NewAIHelper(model_ AIModel, SessionID string) *AIHelper {
	return &AIHelper{
		model:    model_,
		messages: make([]*model.Message, 0),

		// AddMessage が保存を必要とするときに呼ばれる
		saveFunc: func(msg *model.Message) (*model.Message, error) {
			// message を RabbitMQ のメッセージ本文に変換する
			data := rabbitmq.GenerateMessageMQParam(msg.SessionID, msg.Content, msg.UserName, msg.IsUser)

			// RabbitMQ へ publish し、DB 書き込みは consumer が非同期で処理する
			err := rabbitmq.RMQMessage.Publish(data)
			return msg, err
		},

		// 1 つのチャット会話に 1 つの AIHelper を紐づける
		SessionID: SessionID,
	}
}
```

## 3. AIHelper：メッセージをメモリコンテキストへ入れる

ファイル：`GoNexus/common/aihelper/aihelper.go`

```go
func (a *AIHelper) AddMessage(Content string, UserName string, IsUser bool, Save bool) {
	userMsg := model.Message{
		// このメッセージが属する会話
		SessionID: a.SessionID,

		// メッセージ本文
		Content: Content,

		// 現在のユーザー
		UserName: UserName,

		// true はユーザーメッセージ、false は AI メッセージ
		IsUser: IsUser,
	}

	// まず現在の AIHelper のメモリコンテキストへ保存する
	a.messages = append(a.messages, &userMsg)

	// Save が true のときだけ永続化処理へ進む
	if Save {
		a.saveFunc(&userMsg)
	}
}
```

## 4. RabbitMQ：メッセージを消費して DB へ書き込む

ファイル：`GoNexus/common/rabbitmq/meesage.go`

```go
func MQMessage(msg *amqp.Delivery) error {
	var param MessageMQParam

	// RabbitMQ のメッセージ本文を解析する
	err := json.Unmarshal(msg.Body, &param)
	if err != nil {
		return err
	}

	// DB 用の message model に変換する
	newMsg := &model.Message{
		SessionID: param.SessionID,
		Content:   param.Content,
		UserName:  param.UserName,
		IsUser:    param.IsUser,
	}

	// consumer が非同期に DB へ挿入する
	message.CreateMessage(newMsg)
	return nil
}
```

## 5. DAO：メッセージを MySQL へ書き込む

ファイル：`GoNexus/dao/message/message.go`

```go
func CreateMessage(message *model.Message) (*model.Message, error) {
	// GORM でチャットメッセージを 1 件作成する
	err := mysql.DB.Create(message).Error
	return message, err
}
```

## 6. Model：メッセージテーブル構造

ファイル：`GoNexus/model/message.go`

```go
type Message struct {
	// メッセージ主キー
	ID uint `gorm:"primaryKey;autoIncrement" json:"id"`

	// このメッセージが属する会話
	SessionID string `gorm:"index;not null;type:varchar(36)" json:"session_id"`

	// このメッセージが属するユーザー
	UserName string `gorm:"type:varchar(20)" json:"username"`

	// メッセージ本文
	Content string `gorm:"type:text" json:"content"`

	// true はユーザーメッセージ、false は AI メッセージ
	IsUser bool `gorm:"not null;" json:"is_user"`

	CreatedAt time.Time `json:"created_at"`
}
```
