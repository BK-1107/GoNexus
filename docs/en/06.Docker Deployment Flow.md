# 06. Docker Deployment Flow

Docker deployment starts the frontend, backend, MySQL, Redis Stack, and RabbitMQ together.
This chapter explains how `docker-compose.yml` brings up the GoNexus stack and how containers communicate.

## 1. Full Flow

```text
docker-compose up -d
  -> frontend container
  -> backend container
  -> mysql container
  -> redis-stack container
  -> rabbitmq container
  -> backend reads config.toml and environment variables
  -> backend connects to MySQL / Redis / RabbitMQ
  -> frontend Nginx proxies /api to backend:9090
```

Related files:

```text
GoNexus/docker-compose.yml
  -> GoNexus/Dockerfile
  -> GoNexus/frontend/Dockerfile
  -> GoNexus/frontend/nginx.conf
  -> GoNexus/config/config.go
  -> GoNexus/main.go
  -> GoNexus/common/mysql/mysql.go
  -> GoNexus/common/redis/redis.go
  -> GoNexus/common/rabbitmq/rabbitmq.go
```

## 2. Docker Compose: Start All Services

File: `GoNexus/docker-compose.yml`

```yaml
services:
  frontend:
    # Build frontend image
    build:
      context: ./frontend
      dockerfile: Dockerfile
    container_name: gonexus_frontend
    restart: always
    ports:
      # host 8080 -> container 80
      - "8080:80"
    depends_on:
      # frontend waits for backend
      - backend
```

```yaml
  backend:
    # Build Go backend image
    build:
      context: .
      dockerfile: Dockerfile
    container_name: gonexus_backend
    restart: always
    ports:
      # host 9090 -> container 9090
      - "9090:9090"
```

## 3. Backend: Environment Variables

File: `GoNexus/docker-compose.yml`

```yaml
environment:
  # Gin listen address
  GONEXUS_HOST: 0.0.0.0
  GONEXUS_PORT: 9090

  # backend connects to mysql:3306 inside Docker network
  GONEXUS_MYSQL_HOST: mysql
  GONEXUS_MYSQL_PORT: 3306
  GONEXUS_MYSQL_USER: root
  GONEXUS_MYSQL_PASSWORD: root
  GONEXUS_MYSQL_DATABASE: GoNexus

  # Redis Stack service name
  GONEXUS_REDIS_HOST: redis
  GONEXUS_REDIS_PORT: 6379

  # RabbitMQ service name
  GONEXUS_RABBITMQ_HOST: rabbitmq
  GONEXUS_RABBITMQ_PORT: 5672
  GONEXUS_RABBITMQ_USERNAME: root
  GONEXUS_RABBITMQ_PASSWORD: 123456
```

## 4. Backend: Mounted Volumes

File: `GoNexus/docker-compose.yml`

```yaml
volumes:
  # RAG uploaded files are stored on host GoNexus/uploads
  - ./uploads:/app/uploads

  # backend logs are stored on host GoNexus/logs
  - ./logs:/app/logs

  # local config.toml is mounted as read-only
  - ./config/config.toml:/app/config/config.toml:ro
```

## 5. Middleware Containers

File: `GoNexus/docker-compose.yml`

```yaml
mysql:
  image: mysql:8.0
  environment:
    # root password
    MYSQL_ROOT_PASSWORD: root

    # default database
    MYSQL_DATABASE: GoNexus
  volumes:
    # persist MySQL data
    - mysql_data:/var/lib/mysql
```

```yaml
redis:
  image: redis/redis-stack:latest
  ports:
    # Redis port
    - "6379:6379"

    # Redis Stack UI
    - "8001:8001"
  volumes:
    # persist Redis data
    - redis_data:/data
```

```yaml
rabbitmq:
  image: rabbitmq:3-management
  environment:
    # RabbitMQ login
    RABBITMQ_DEFAULT_USER: root
    RABBITMQ_DEFAULT_PASS: 123456
  ports:
    # AMQP port
    - "5672:5672"

    # management UI
    - "15672:15672"
  volumes:
    # persist RabbitMQ data
    - rabbitmq_data:/var/lib/rabbitmq
```

## 6. Backend Dockerfile: Build Go Service

File: `GoNexus/Dockerfile`

```dockerfile
FROM golang:1.24-bookworm AS builder

WORKDIR /src

# Copy go.mod and go.sum first to cache dependency download
COPY go.mod go.sum ./
RUN go mod download

# Copy backend source code
COPY . .

# Build Linux binary
RUN CGO_ENABLED=0 GOOS=linux go build -tags=nomsgpack -trimpath -ldflags="-s -w -checklinkname=0" -o /out/gonexus .
```

```dockerfile
FROM gcr.io/distroless/static-debian12:nonroot

WORKDIR /app

# Copy only compiled binary into runtime image
COPY --from=builder /out/gonexus /app/gonexus

EXPOSE 9090

# Run backend service when container starts
ENTRYPOINT ["/app/gonexus"]
```

## 7. Frontend Dockerfile and Nginx Proxy

Files: `GoNexus/frontend/Dockerfile`, `GoNexus/frontend/nginx.conf`

```dockerfile
FROM node:24-alpine AS builder

WORKDIR /app

# Install frontend dependencies
COPY package*.json ./
RUN npm ci

# Build dist files
COPY . .
RUN npm run build
```

```dockerfile
FROM nginx:1.29-alpine

# Use custom Nginx config
COPY nginx.conf /etc/nginx/conf.d/default.conf

# Copy Vite output into Nginx static directory
COPY --from=builder /app/dist /usr/share/nginx/html
```

```nginx
location /api/ {
    # Compose internal network uses service name backend
    proxy_pass http://backend:9090/api/;

    # Disable buffering for SSE
    proxy_buffering off;

    # Streaming chat may last longer
    proxy_read_timeout 3600s;
}
```

## 8. Config and Backend Startup

Files: `GoNexus/config/config.go`, `GoNexus/main.go`

```go
func InitConfig() error {
	// Default config path
	configPath := os.Getenv("GONEXUS_CONFIG_PATH")
	if configPath == "" {
		configPath = "config/config.toml"
	}

	// Load config.toml first when it exists
	if _, err := os.Stat(configPath); err == nil {
		if _, err := toml.DecodeFile(configPath, config); err != nil {
			return err
		}
	}

	// Then override with environment variables
	applyEnvOverrides(config)
	return nil
}
```

```go
func main() {
	// Initialize logs
	logFile, err := initLogging()
	if err != nil {
		return
	}
	defer logFile.Close()

	// Read config
	conf := config.GetConfig()

	// Initialize MySQL and AutoMigrate
	mysql.InitMysql()

	// Initialize Redis
	redis.Init()

	// Initialize RabbitMQ and start consumer
	rabbitmq.InitRabbitMQ()

	// Start Gin HTTP server
	StartServer(conf.MainConfig.Host, conf.MainConfig.Port)
}
```

## 9. Commands

```bash
# Enter backend project directory
cd GoNexus

# Build and start all containers in background
docker-compose up -d

# Check container status
docker-compose ps

# Follow backend logs
docker-compose logs -f backend

# Stop all containers
docker-compose down
```
