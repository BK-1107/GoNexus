# 05. Image Recognition Flow

Image recognition sends an uploaded image from the frontend to an OpenAI-compatible Vision API.
This chapter explains the flow from the `/vision` page to backend model invocation and result display.

## 1. Full Flow

```text
/vision page
  -> ImageRecognition.tsx
  -> imageApi.recognize
  -> POST /api/v1/image/recognize
  -> Router
  -> controller
  -> service
  -> image to base64 data URL
  -> OpenAI-compatible Vision API
  -> controller returns class_name
  -> frontend displays result
```

Related files:

```text
GoNexus/frontend/src/App.tsx
  -> GoNexus/frontend/src/pages/VisionPage.tsx
  -> GoNexus/frontend/src/components/chat/ImageRecognition.tsx
  -> GoNexus/frontend/src/api/image.ts
  -> GoNexus/router/Image.go
  -> GoNexus/controller/image/image.go
  -> GoNexus/service/image/image.go
  -> GoNexus/config/config.go
```

const handleStartAnalysis = async () => {

  // Enter analyzing state
  setStatus('analyzing')
  try {
    // Call backend image recognition API

      // Store backend recognition result
      setBackendResult(res.data.class_name)
      setStatus('done')
    } else {
      setStatus('error')
    }
  } catch (err) {
    setStatus('error')
  }
}

```go
type RecognizeImageResponse struct {
	// Image description or recognition result returned by AI vision model
	ClassName string `json:"class_name,omitempty"`
	controller.Response
}
```

```go
func RecognizeImage(c *gin.Context) {
	res := new(RecognizeImageResponse)

	// Read image from multipart form field image
	file, err := c.FormFile("image")
	if err != nil {
		c.JSON(http.StatusOK, res.CodeOf(code.CodeInvalidParams))
		return
	}

	// Let service call vision model
	className, err := image.RecognizeImage(file)
	if err != nil {
		c.JSON(http.StatusOK, res.CodeOf(code.CodeServerBusy))
		return
	}

	// Return recognition result
	res.Success()
	res.ClassName = className
	c.JSON(http.StatusOK, res)
}
```

## 5. Service: Read Image Content

File: `GoNexus/service/image/image.go`

```go
func RecognizeImage(file *multipart.FileHeader) (string, error) {
	// Open uploaded image
	src, err := file.Open()
	if err != nil {
		return "", fmt.Errorf("failed to open image: %v", err)
	}
	defer src.Close()

	// Read image binary data
	imgData, err := io.ReadAll(src)
	if err != nil {
		return "", fmt.Errorf("failed to read image data: %v", err)
	}
```

## 6. Service: Convert to base64 data URL

File: `GoNexus/service/image/image.go`

```go
ext := strings.ToLower(filepath.Ext(file.Filename))
mimeType := "image/jpeg"

// Set image MIME type by extension
if ext == ".png" {
	mimeType = "image/png"
} else if ext == ".webp" {
	mimeType = "image/webp"
}

// Convert image to data URL accepted by OpenAI-compatible API
dataURL := fmt.Sprintf("data:%s;base64,%s", mimeType, base64.StdEncoding.EncodeToString(imgData))
```

## 9. Service: Request Vision API

File: `GoNexus/service/image/image.go`

```go
// OpenAI-compatible endpoint
apiEndpoint := strings.TrimRight(baseURL, "/") + "/chat/completions"

// Create HTTP POST request
req, err := http.NewRequest("POST", apiEndpoint, bytes.NewBuffer(jsonPayload))
if err != nil {
	return "", fmt.Errorf("failed to create HTTP request: %v", err)
}

// Set JSON and Bearer token
req.Header.Set("Content-Type", "application/json")
req.Header.Set("Authorization", "Bearer "+apiKey)

// Call vision model API
resp, err := http.DefaultClient.Do(req)
if err != nil {
	return "", fmt.Errorf("failed to call AI Vision API: %v", err)
}
defer resp.Body.Close()
```

## 10. Service: Parse Model Response

File: `GoNexus/service/image/image.go`

```go
// Read response body
respBody, err := io.ReadAll(resp.Body)
if err != nil {
	return "", fmt.Errorf("failed to read API response: %v", err)
}

var aiResp AIResponse

// Parse OpenAI-compatible response JSON
if err := json.Unmarshal(respBody, &aiResp); err != nil {
	return "", fmt.Errorf("failed to parse AI response JSON: %v", err)
}

// Model API returned error
if aiResp.Error != nil {
	return "", fmt.Errorf("AI API Error: %s", aiResp.Error.Message)
}

// Return first answer
if len(aiResp.Choices) > 0 {
	return aiResp.Choices[0].Message.Content, nil
}
```
