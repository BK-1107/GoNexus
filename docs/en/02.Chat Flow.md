# 02. Chat Flow

This chapter focuses on the current main frontend path: SSE streaming chat.

## 1. Full Flow

```text
ChatInput.tsx
  -> useStreaming.ts
  -> POST /api/v1/AI/chat/send-stream or /send-stream-new-session
  -> Router
  -> JWT middleware
  -> controller
  -> service
  -> AIHelperManager
  -> AIModelFactory
  -> AIHelper
  -> AIModel.StreamResponse
  -> SSE data: xxx\n\n returns to frontend
  -> frontend updateLastMessage
  -> RabbitMQ
  -> dao/message
  -> MySQL
```

Related files:

```text
GoNexus/frontend/src/components/chat/ChatInput.tsx
  -> GoNexus/frontend/src/hooks/useStreaming.ts
```

```go
		// New session: create session first, then stream AI response
```

## 8. Service: Create New Chat Session

File: `GoNexus/service/session/session.go`

```go
func CreateStreamSessionOnly(userName string, userQuestion string) (string, code.Code) {
	newSession := &model.Session{
		// Unique ID for each chat window
		ID: uuid.New().String(),

		// Bind current login user
		UserName: userName,

		// Use the first question as default title
		Title: userQuestion,
	}

	// Write session through dao
	createdSession, err := daosession.CreateSession(newSession)
	if err != nil {
		return "", code.CodeServerBusy
	}

	// Return new sessionId
	return createdSession.ID, code.CodeSuccess
}
```

```go
	_, err := writer.Write([]byte("data: " + msg + "\n\n"))
	if err != nil {
		return
	}
```

## 11. Frontend: Update Last AI Message

File: `GoNexus/frontend/src/store/chatStore.ts`

```ts
updateLastMessage: (content) => set((state) => {
  const newMessages = [...state.messages]
  if (newMessages.length > 0) {
    // The last message is the empty AI message inserted earlier
    newMessages[newMessages.length - 1] = { 
      ...newMessages[newMessages.length - 1], 
      // Replace it with latest accumulated content
      content 
    }
  }
  return { messages: newMessages }
})
```
