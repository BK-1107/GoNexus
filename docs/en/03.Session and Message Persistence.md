# 03. Session and Message Persistence

Session and message persistence stores user messages and AI replies.
This chapter explains how AIHelper first stores messages in memory context, then asynchronously writes them to MySQL through RabbitMQ.

## 1. Full Flow

```text
AIHelper.AddMessage
  -> in-memory context a.messages
  -> saveFunc
  -> GenerateMessageMQParam
  -> RabbitMQ Publish
  -> MQMessage consumer
  -> dao/message
  -> MySQL
```

Related files:

```text
GoNexus/common/aihelper/aihelper.go
  -> GoNexus/common/rabbitmq/meesage.go
  -> GoNexus/common/rabbitmq/rabbitmq.go
  -> GoNexus/dao/message/message.go
  -> GoNexus/common/mysql/mysql.go
  -> GoNexus/model/message.go
```

## 2. AIHelper: Create Save Callback

File: `GoNexus/common/aihelper/aihelper.go`

```go
func NewAIHelper(model_ AIModel, SessionID string) *AIHelper {
	return &AIHelper{
		model:    model_,
		messages: make([]*model.Message, 0),

		// Called when AddMessage needs persistence
		saveFunc: func(msg *model.Message) (*model.Message, error) {
			// Convert message into RabbitMQ payload
			data := rabbitmq.GenerateMessageMQParam(msg.SessionID, msg.Content, msg.UserName, msg.IsUser)

			// Publish to RabbitMQ; database writing is handled asynchronously by consumer
			err := rabbitmq.RMQMessage.Publish(data)
			return msg, err
		},

		// One chat session binds to one AIHelper
		SessionID: SessionID,
	}
}
```

## 3. AIHelper: Message Enters Memory Context

File: `GoNexus/common/aihelper/aihelper.go`

```go
func (a *AIHelper) AddMessage(Content string, UserName string, IsUser bool, Save bool) {
	userMsg := model.Message{
		// Chat session this message belongs to
		SessionID: a.SessionID,

		// Message body
		Content: Content,

		// Current user
		UserName: UserName,

		// true means user message, false means AI message
		IsUser: IsUser,
	}

	// Store in current AIHelper memory context first
	a.messages = append(a.messages, &userMsg)

	// Only persist when Save is true
	if Save {
		a.saveFunc(&userMsg)
	}
}
```

## 4. AIHelper: Save User Message and AI Reply

File: `GoNexus/common/aihelper/aihelper.go`

```go
func (a *AIHelper) StreamResponse(userName string, ctx context.Context, cb StreamCallback, userQuestion string) (*model.Message, error) {
	// Save user question
	a.AddMessage(userQuestion, userName, true, true)

	a.mu.RLock()
	messages := utils.ConvertToSchemaMessages(a.messages)
	a.mu.RUnlock()

	// Generate AI reply
	content, err := a.model.StreamResponse(ctx, messages, cb)
	if err != nil {
		return nil, err
	}

	modelMsg := &model.Message{
		SessionID: a.SessionID,
		UserName:  userName,
		Content:   content,
		IsUser:    false,
	}

	// Save AI reply
	a.AddMessage(modelMsg.Content, userName, false, true)

	return modelMsg, nil
}
```

## 5. RabbitMQ: Generate Payload

File: `GoNexus/common/rabbitmq/meesage.go`

```go
type MessageMQParam struct {
	// Session ID
	SessionID string `json:"session_id"`

	// Message content
	Content string `json:"content"`

	// Username
	UserName string `json:"user_name"`

	// Whether it is a user message
	IsUser bool `json:"is_user"`
}
```

```go
func GenerateMessageMQParam(sessionID string, content string, userName string, IsUser bool) []byte {
	param := MessageMQParam{
		SessionID: sessionID,
		Content:   content,
		UserName:  userName,
		IsUser:    IsUser,
	}

	// Convert to JSON bytes as RabbitMQ body
	data, _ := json.Marshal(param)
	return data
}
```

## 6. RabbitMQ: Consume and Write Database

File: `GoNexus/common/rabbitmq/meesage.go`

```go
func MQMessage(msg *amqp.Delivery) error {
	var param MessageMQParam

	// Parse RabbitMQ message body
	err := json.Unmarshal(msg.Body, &param)
	if err != nil {
		return err
	}

	// Convert to database message model
	newMsg := &model.Message{
		SessionID: param.SessionID,
		Content:   param.Content,
		UserName:  param.UserName,
		IsUser:    param.IsUser,
	}

	// Consumer inserts into database asynchronously
	message.CreateMessage(newMsg)
	return nil
}
```

## 8. Model: Message Table Structure

type Message struct {
	// Message primary key
	ID uint `gorm:"primaryKey;autoIncrement" json:"id"`

	// Chat session this message belongs to
	SessionID string `gorm:"index;not null;type:varchar(36)" json:"session_id"`

	// User this message belongs to
	UserName string `gorm:"type:varchar(20)" json:"username"`

	// Message body
	Content string `gorm:"type:text" json:"content"`

	// true means user message, false means AI message
	IsUser bool `gorm:"not null;" json:"is_user"`

	CreatedAt time.Time `json:"created_at"`
}
