# 01. User Authentication

User authentication covers registration, login, captcha, and invite-code validation.
This chapter focuses on the login flow: how the backend validates credentials, generates a JWT, and lets the frontend keep the login state.

## 1. Full Flow

```text
POST /api/v1/user/login
  -> Router
  -> controller
  -> service
  -> dao
  -> MySQL
  -> service generates JWT
  -> controller returns token
  -> frontend stores token
```

Related files:

```text
GoNexus/frontend/src/pages/AuthPage.tsx
  -> GoNexus/router/user.go
  -> GoNexus/controller/user/user.go
  -> GoNexus/service/user/user.go
  -> GoNexus/dao/user/user.go
  -> GoNexus/common/mysql/mysql.go
  -> GoNexus/model/user.go
  -> GoNexus/utils/myjwt/jwt.go
  -> GoNexus/frontend/src/store/authStore.ts
```

## 2. POST: Frontend Sends Login Request

File: `GoNexus/frontend/src/pages/AuthPage.tsx`

```tsx
// Login uses /user/login, registration uses /user/register
const endpoint = isLogin ? "/user/login" : "/user/register"

// Login needs username and password; registration needs email, password, and captcha
const payload = isLogin ? { username, password } : { email, password, captcha }

// Send request to backend
const res = await axios.post(apiUrl(endpoint), payload)
```

Actual login request:

```tsx
axios.post(apiUrl("/user/login"), {
  // Login username entered by the user
  username,

  // Login password entered by the user
  password,
})
```

```go
func Login(c *gin.Context) {
	req := new(LoginRequest)
	res := new(LoginResponse)

	// Bind frontend JSON body into req
	if err := c.ShouldBindJSON(req); err != nil {
		c.JSON(http.StatusOK, res.CodeOf(code.CodeInvalidParams))
		return
	}

	// Validate credentials and generate token
	token, code_ := user.Login(req.Username, req.Password)
	if code_ != code.CodeSuccess {
		c.JSON(http.StatusOK, res.CodeOf(code_))
		return
	}

	// Return token to frontend
	res.Success()
	res.Token = token
	c.JSON(http.StatusOK, res)
}
```

## 5. Service: Validate Username and Password

File: `GoNexus/service/user/user.go`

```go
func Login(username, password string) (string, code.Code) {
	var userInformation *model.User
	var ok bool

	// Check whether the user exists
	if ok, userInformation = user.IsExistUser(username); !ok {
		return "", code.CodeUserNotExist
	}

	// Hash the input password with MD5 and compare it with database password
	if userInformation.Password != utils.MD5(password) {
		return "", code.CodeInvalidPassword
	}

	// Credentials are valid, generate JWT token
	token, err := myjwt.GenerateToken(userInformation.ID, userInformation.Username)
	if err != nil {
		return "", code.CodeServerBusy
	}

	// Return token to controller
	return token, code.CodeSuccess
}
```

func GetUserByUsername(username string) (*model.User, error) {
	user := new(model.User)

	// GORM queries the first record whose username matches
	err := DB.Where("username = ?", username).First(user).Error
	return user, err
}

## 10. JWT Utility: Generate Token

File: `GoNexus/utils/myjwt/jwt.go`

```go
func GenerateToken(id uint, username string) (string, error) {
	claims := MyClaims{
		// User ID
		ID: id,

		// Username
		Username: username,

		RegisteredClaims: jwt.RegisteredClaims{
			// Token issuer
			Issuer: config.GetConfig().Issuer,

			// Token subject
			Subject: config.GetConfig().Subject,

			// Token expiration time
			ExpiresAt: jwt.NewNumericDate(time.Now().Add(time.Hour * time.Duration(config.GetConfig().ExpireDuration))),

			// Token issued time
			IssuedAt: jwt.NewNumericDate(time.Now()),
		},
	}

	// Create token using HS256
	token := jwt.NewWithClaims(jwt.SigningMethodHS256, claims)

	// Sign with configured key and return token string
	return token.SignedString([]byte(config.GetConfig().Key))
}
```
